# Password Strength Analyzer

## Project Overview

This project is a **Password Strength Analyzer** built using Python. The goal is to help users evaluate the strength of their passwords based on key security criteria. This tool is a basic example of **security automation**, providing quick feedback on password quality to encourage stronger password creation.

## Features

- Prompts user to enter a password securely
- Evaluates:
  - Password length
  - Use of uppercase and lowercase letters
  - Presence of numbers and symbols
- Provides a strength rating:
  - Weak
  - Moderate
  - Strong
- Displays actionable feedback

2. Open Terminal or PowerShell
